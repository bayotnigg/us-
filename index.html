INTRODUCTION TO JAVASCRIPT

JavaScript is one of the most popular coding languages in the world, it can be used to provide functionality to webpages, and make applications for many things. A JavaScript Developer makes around 26k-81k Philippine Pesos a month in the Philippines.

JavaScript has no link to the coding language Java whatsoever and conforms to the ECMAScript standard, a standard for scripting languages. First created and implemented in the really old browser, Netscape Navigator in 1994. An outstanding web browser for its time.

BASICS IN JAVASCRIPT

To write JavaScript code, you must have a text editor / code editor. A popular code editor is Visual Studio Code
(where this website was tested and written in before being injected into Trinket.io).

Create a new file in VSCode, (name).html and then another file named (name).js then do ! + tab to create a html boiler plate and link the js file to the html file by placing a script tag in the head tag or in the body tag then link the js file with src="name.js".
